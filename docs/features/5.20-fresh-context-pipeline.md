## A. Vấn Đề Hiện Tại

### 1. Stale Context Cache (Gap E)
`ContextLoadStep` chạy một lần duy nhất lúc khởi tạo task. Trong `builder.go:701`, nếu `cachedData != nil`, assembler bỏ qua `ctxEngine.RetrieveContext`. Cache chứa snippets từ trước khi code được viết.

### 2. Host Absolute Path Leaks (Gap F)
`IndexWorkspace` scan toàn bộ `code/repos`. Khi `ToLogical` throw boundary error, fallback dùng raw absolute host path (`provider.go:434`), rò rỉ cấu trúc host filesystem cho LLM.
